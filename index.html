Timestamp Microservice

Enter a string in the URL, and this service will determine if it is a UNIX or natural language date. If it is, the date will be presented back to you in both formats. If it isn't, the program will return null.

For Example:

  • http://localhost:3000/1265094000000
  • http://localhost:3000/February 2, 2010
  • http://localhost:3000/February%202,%202010
